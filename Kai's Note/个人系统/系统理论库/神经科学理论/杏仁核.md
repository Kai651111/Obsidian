# 1 在大脑中的位置和地位
- 位置：Limbic System（哺乳脑）里，旁边就是新皮质（人类脑）。
- 地位：大脑的“威胁雷达 + 情绪触发器”，负责在极短时间内判断：**现在是否危险？是否需要立刻反应？**
# 2 杏仁核的功能
- 危险的快速识别：通过SAM轴反应。
- 情绪放大：通过SAM+HPA轴的反应、产生皮质醇（战或逃）
- 和海马体协作：创伤事件记忆深刻。
# 3 杏仁核的过度敏感
## 3.1 表现
- 小刺激导致大情绪
- 不确定性高度担忧
- 身体很难放松，很紧张
- 睡不着
## 3.2 有效训练
- 提高可控**不确定性暴漏量**：多暴漏在社交场合里。
- 深呼吸。
	- 3步呼吸法：吸满、再吸、呼气。
- 节律性运动（中低强度）
- 使用PFC控制：
	- 明明情绪（这是焦虑，不是危险）
	- 延迟反应90s
	- 理智分析
- 睡眠与节律训练
	- 睡眠不足，杏仁核敏感性大幅度提升。
