# 2 皮质醇的优点
## 2.1 皮质醇的优点
- 提高能量（短期爆发力）：
	- 促进糖异生，提高血糖
	- 降低脂肪和蛋白质——转化为游离的脂肪酸和氨基酸
- 提升警觉和注意力——肾上腺素
- 可以增强血管对NE的反应，提升前额叶的警觉性：
　　　　　　![|475](assets/个人系统/系统理论库/神经科学理论/HPA轴与皮质醇/Pasted%20image%2020260210200043.png)
- 抑制炎症反应
- 强化神经元的短期连接
## 2.2 皮质醇的缺点
- 对大脑的长期伤害
	- 损害海马体。
	- 杏仁核会变大、变敏感，降低PFC的控制。
		- REM睡眠不足会让杏仁核敏感。
- 抑制了“短期非必要系统，比如：
	- 免疫系统（抑制NK细胞、B细胞、T细胞，降低免疫系统的敏感性）
	- 生殖系统（睾酮和皮质醇是拮抗作用）
	- 消化系统。
- 脂肪重新分布（腹部脂肪）
- 通过刺激LC-NA系统，影响睡眠
	- 失眠：皮质醇峰值错位。
	- 早醒+晨起累：皮质醇堆积过多。
	- 慢性皮质醇偏高：减少深睡和REM，增加夜间清醒次数。


# 3 HPA轴与皮质醇——Characteristic & Application
## 3.1 与SAM轴对比：HPA轴的滞后性
‌‌‌‌　　HPA轴的一个特性，是其具有滞后性（相对于SAM轴来讲）。
‌‌‌‌　　HPA轴是靠激素调节（下丘脑-垂体-肾上腺皮质），所以肯定赶不上电信号调节的SAM轴（交感神经兴奋）。

‌‌‌‌产生的**瞬时焦虑**：
- SAM轴快速反应。
- HPA轴还没完全激活，皮质醇的产生具有滞后性（10-20min上升，30-60min达到峰值）
- 可以利用这个时间差，来控制皮质醇的产生。

## 3.2 皮质醇节律
### 3.2.1 正常的一天：皮质醇的变化
- 睡觉前：皮质醇应该最低，感到困倦。
- 凌晨3-5点：明显升高。
- 早上起床：最高，可以唤醒一个人。

### 3.2.2 非正常的皮质醇变化
- 早上起来脑雾：
	- HPA轴混乱，皮质醇峰值错位
	- `仅仅用七天就可以重置HPA轴。`
- 早醒/失眠：
	- 皮质醇的基础水平偏高。

## 3.3 提高与降低皮质醇
### 3.3.1 早上提高皮质醇
- 接触阳光
	- 视网膜SCN（视交叉上核）重置昼夜节律，容易分泌褪黑素
- 提高LC-NE的活性（产生NE）
	- 需要运动。
	- 看这个：[神经科学原理：运动、压力与意志力 26-02-07.excalidraw](ExcalidrawSave/神经科学原理：运动、压力与意志力%2026-02-07.excalidraw.md)
- 补充水分
- 咖啡因：延长皮质醇的作用时间
	- 但是不要刚醒就喝，等一下
- 冷暴露。

### 3.3.2 晚上降低皮质醇
- 睡前摄入碳水
- 呼吸练习
- 避免高心理激活行为
- 下午就不要摄入咖啡因了
- 白天晒够太阳
- 避免照射蓝光：
	- 手机的蓝光严重影响睡眠。
	- 灯光太亮也会。
- 南非醉茄
	- 坚持一个月，显著降低皮质醇。
