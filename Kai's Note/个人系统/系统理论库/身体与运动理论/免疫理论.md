# 2 睡眠——生长激素与免疫细胞（NK）
- 在深睡眠中，皮质醇下降、褪黑素与生长激素上升，使免疫细胞得以修复、增殖并形成免疫记忆；  
- 睡眠不足会导致慢性炎症升高、免疫反应失调（杀死一大半的NK细胞），使身体长期处于“应激而非防御”状态，显著降低抗感染与恢复能力。

# 3 运动：
## 3.1 适度运动
- **适度运动**通过**触发SAM轴**、**短暂激活免疫系统**并降低慢性炎症基线，显著提升免疫功能。  
	- 在**适度运动**中，发生的是三件事
		1. 交感神经 + 去甲肾上腺素（NE）↑
		2. 免疫细胞被“动员”进入血液循环
		3. 免疫监测效率瞬间提升
	- 这个过程叫：**immune surveillance boost（免疫巡逻增强）**
- 低强度运动：
	- zone2可以增强免疫功能。

## 3.2 过度运动与免疫开窗
### 3.2.1 过量皮质醇抑制免疫反应
- 皮质醇可以抑制免疫系统功能（他还抑制免疫反应）
- 运动的时候会产生皮质醇。
- 由于皮质醇的滞后性，如果运动时间过长、太超负荷，免疫系统就会被抑制（NK、B细胞与皮质醇拮抗）。
- 这一效应也被称为：”**免疫开窗效应“**。
### 3.2.2 免疫开窗效应的应用
- 科学训练安排：
	- 避免通宵、长途旅行、睡眠不足的情况下，再次高强度训练。
- 及时补充碳水：
	- 运动后补充碳水可降低皮质醇反应。
	- ***赶紧来一根香蕉！***
- 避免再次高强度刺激
	- 高强度训练的间隔：24-72小时。

# 4 淋巴循环与久坐
## 4.1 原理：
- 血液循环可以帮我们的血液循环，而淋巴循环没有泵、单向阀（靠肌肉挤）、流速慢。
- 淋巴循环是免疫系统的载体，所以需要通过**主动地**去进行淋巴循环。
## 4.2 提升淋巴循环
- 深呼吸
	- 横膈膜上下运动，胸腔被挤压，全身淋巴回流。
- 多关节活动
	- 走路、瑜伽、跳绳、游泳、汽车等等
	- 淋巴循环需要肌肉的作用，而多关节活动肌肉的活动更多。
- 重力变化帮助下肢淋巴回流
	- 比如轻度倒立、压腿、平躺+深呼吸一起做、瑜伽等等
- 按摩与拉伸
	- 脸部与颈部的淋巴：可以通过拉伸和按摩来完成。
	- 其他部位的淋巴：筋膜枪、按摩、拉伸、泡沫轴
- 冷热交替
	- 温水澡：血管扩张，利于流动
	- 冷刺激：短暂收缩后反弹
